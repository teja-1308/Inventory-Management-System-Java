# Inventory Management System - Core Java

A console-based inventory management application built using **Core Java** with file persistence.

## 📌 Features

- **Add Product**: Add new products with ID, Name, Quantity, and Price
- **View Products**: Display all products in inventory
- **Update Product**: Modify existing product details
- **Delete Product**: Remove products from inventory
- **Search Product**: Find products by ID
- **Data Persistence**: Data is saved to `inventory.txt` file so it won't be lost after closing the application
- **Exception Handling**: Handles invalid inputs and prevents program crashes

## 🛠️ Technologies Used

- **Language**: Java
- **Concepts**: OOP, Encapsulation, Collections, File I/O, Exception Handling
- **Tools**: VS Code, Git, GitHub

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/teja-1308/Inventory-Management-System-Java.git
   ```
2. **Compile the Java files**
    ```bash
   javac Main.java 
      java Main
   ```
3. **Run the application**
   ```bash
   java Main
   ```
